# Product Image Placeholders

Bu klasore urun gorsellerinin placeholder dosyalari eklenmelidir.

Gerekli ana gorseller:
- `hero.webp`
- `angle-1.webp`
- `angle-2.webp`
- `angle-3.webp`
- `lifestyle.webp`

Boyut ve format:
- Ana urun gorselleri: `800x800px`
- Tercih edilen format: `WebP`
- Alternatif format: `JPG`

Open Graph gorseli:
- `og-image.jpg`
- Boyut: `1200x630px`
